# CluePilot AI 智能线索推理助手

CluePilot AI 是一个基于 Web 前端实现的轻量级推理交互项目。项目模拟 AI Agent 的“信息感知—线索分析—结论生成”流程，用户可以阅读案件背景、嫌疑人信息和关键线索，并判断最可能的嫌疑人。

# 项目功能

- 随机生成不同案件
- 展示案件背景、嫌疑人信息和关键线索
- 支持用户选择嫌疑人并提交判断
- 根据用户选择生成推理反馈
- 展示正确答案、推理原因和改进建议
- 页面支持响应式布局，适配电脑和移动端

## 技术栈

- HTML
- CSS
- JavaScript

# 项目亮点

- 使用结构化数据管理案件信息
- 将案件拆分为背景、人物、线索、答案和推理原因
- 模拟 AI Agent 的基础执行流程
- 具备较强的交互性和展示性
- 适合作为 AI Native / 前端入门项目展示

## AI Agent 思路

本项目虽然没有直接调用大模型 API，但通过规则推理和模板化反馈，模拟了轻量级 Agent 的核心流程：

1. 感知输入：读取案件背景和用户选择
2. 分析信息：比对人物、时间线和关键线索
3. 生成结论：判断用户选择是否正确
4. 输出反馈：生成推理说明和改进建议
